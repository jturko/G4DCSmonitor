Simulation macro/results which test the geometric biasing implementation for a 2 MeV gamma-ray source within a CASTOR fuel assembly (#18, closest to the CLYC). We compare 1e8 gamma rays emitted with either i) no directional bias, or ii) with a directional bias using a stated bounding radius around the CLYC detector. Deviations in the low-energy background can be seen for the smaller bounding radii, suggesting that a non-insignificant contribution to the low-energy background arises from gamma rays which are not emitted directly towards the CLYC detector, and instead arise from scattering within the cask. It appears that, for 2 MeV gammas, that 50-100 cm is a sufficient bounding radius to get below 10% deviations in the integral count number.
